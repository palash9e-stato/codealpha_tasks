## Task 3: Handwritten Character Recognition

This project aims to recognize handwritten digits and characters using deep learning techniques. Convolutional Neural Networks (CNNs) were trained on the MNIST and EMNIST datasets after image preprocessing and normalization. The model learns spatial features to accurately classify characters. The project can be extended to word or sentence recognition using sequence-based models such as CRNN.

## Task 4: Disease Prediction from Medical Data

This project predicts the likelihood of diseases based on structured medical data including age, symptoms, and clinical test results. Multiple machine learning algorithms such as Logistic Regression, SVM, Random Forest, and XGBoost were explored. The model performance was evaluated using accuracy and classification metrics. Public datasets from the UCI Machine Learning Repository were used to ensure real-world relevance.
